学习路线

建议先走完基础路径，再选择一个方向深入。

01 基础准备

Python、线性代数、概率统计、数据处理。

02 机器学习

监督/非监督学习，模型评估，特征工程。

03 深度学习

CNN、RNN、Transformer 与训练技巧。

04 大模型应用

Prompt、RAG、评测、推理与应用编排。

05 Agent 系统

工具调用、记忆、规划执行与多 Agent 协作。

06 Vibe Coding

从想法到原型，快速迭代、验证并上线。
